# My-Dark-Notepad
  веб-приложение "My Dark Notepad" — тёмный блокнот с множеством полезных функций для редактирования, поиска, замены и сохранения текста. Вот описание его основных возможностей:

Интуитивный интерфейс:
Тёмный дизайн с комфортной читаемостью.
Заголовок и логотип в верхней части.
Поле поиска и замены текста с быстрым доступом.
Большая текстовая область для написания и редактирования заметок.
Набор кнопок для сохранения файла в различных форматах.
Возможность загрузки файла для редактирования.
Поиск и замена текста:
Ввод слова для поиска по тексту.
Быстрый переход к следующему совпадению.
Замена всех найденных вхождений на заданное слово.
Сохранение заметок в различных форматах:
.TXT — обычный текст.
.HTML и .HTM — HTML-документ с содержимым.
.BAT — пакетный файл с выводом текста.
.CSS, .JS, .JSON, .PHP, .TS, .SCSS, .SASS, .LESS, .MD — соответствующие форматы с добавленными комментариями или структурой.
.XML — XML-документ.
Загрузка файла:
Возможность выбрать файл со своего устройства и загрузить его содержимое в редактор для дальнейшего редактирования.
Дополнительные функции:
Автоматическая обработка содержимого при сохранении в разные форматы.
Эффекты при наведении (подчеркивание, масштабирование кнопок).
Адаптивный дизайн для различных устройств и разрешений.
Это приложение удобно для хранения, поиска, редактирования и сохранения заметок в различных форматах, подходит для программирования, заметок или быстрой записи идей.
